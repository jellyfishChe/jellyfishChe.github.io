# 麻將記分表 (前端)

簡單的單頁前端應用，用來記錄麻將胡牌的相對積點（相對輸贏）。

使用方法

- 開啟 `index.html`（瀏覽器）。
- 編輯玩家名稱（預設：東/南/西/北，預設點數 0）。
- 預設啟用「相對積點制」，每次記錄會直接對玩家做加減（相對變動）。
- 在「記錄胡牌」表單選擇贏家、付款者並輸入要變動的點數，按「記錄」。
- 自摸（勾選「自摸」）會由其他三家各付該點數給贏家。
- 本版固定採用「相對積點制」，不再提供模式切換。
- 支援復原最後一筆、重設遊戲、匯出 JSON。

資料會儲存在瀏覽器的 `localStorage`。

## 台數表（可自訂）

預設台數/台型清單放在 [mjList.js](mjList.js)。

- 你可以直接編輯 [mjList.js](mjList.js) 來新增、刪除或調整台數。
- 畫面上的「台數」會把你勾選的項目加總，並在每筆「紀錄」中一併保存。
